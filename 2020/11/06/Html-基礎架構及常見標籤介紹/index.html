Html 基礎架構及常見標籤介紹

Html是什麼？

  • HyperText Markup Language

Html是一種基礎的技術，常用來進行網頁設計，Html可以讓瀏覽器讀取，進而呈現出視覺化的介面。

Html可以標示出網頁的標題、段落、列表、表單等，或是透過語意化的標籤來讓網頁更加直觀讓搜尋引擎更容易辨識等等。

Html檔案內也可以嵌入圖像或其他物件，並可以搭配CSS和Javascript創造出更為豐富的網頁介面。

Html是網頁的結構
CSS是網頁的外貌和排版
Javascript是網頁的行為

Html的基本架構

以下這個圖包含了Html檔案內元素的結構


(Image from wikipedia)

一個元素(Element)由起始標籤(Start tag / Open tag)、內容(Content)、結束標籤(End tag / Close tag)組成。

而元素若含有屬性(Attribute)則會寫在起始標籤內，
一個完整的屬性會有屬性名稱(Attribute Name)、等號(=)、屬性值(Attribute Value)

以下是一個Html檔案的基本結構，可以發現其中的Tag也都是兩兩一組，一為起始，一為結束。

第一行的<!DOCTYPE html>是一個宣告，宣告此檔案是用Html格式寫成的，讓瀏覽器用Html格式來讀取這個檔案。

第二跟第九行的<html>以及</html>是這整個網頁的根元素，包含了整個網頁的元素，而其中的文字則描述了整個網頁。

第三跟第五行的<head> </head>包含了
第四行的<title> </title>：網頁標題(在瀏覽器分頁欄看見的文字)
而一些外部代碼如：
CSS：使用<style> </style>標籤
Javascript：使用<script> </script>標籤
也會寫在<head> </head>內。

第六和第八行的<body> </body>則是網頁的內容，
其中在第七行的<p> </p>代表了一個段落。

Html常用標籤介紹

  • 標題<h1>~ <h6>

    字體會由大至小變化

  • 段落<p> </p>

    有序列表是有順序的列出其中的內容，內容由<li> </li>內的文字決定。

  • 無序列表：Unordered Lists <ul> </ul> <li>

    無序列表則是沒有順序的列出其中的內容，內容由<li> </li>內的文字決定。

  • 圖片：Image <img>
