金魚切版練習 − 通用橫式版面

Jesse Fang
2020-11-12
CSS

金魚切版練習5鐵人賽網頁傳送門

首先我們來看看實際做出來的畫面呈現↓
實作完成傳送門

可以看到畫面上有三個橫向排列並包含有圖片及文字的區塊
同時透過簡易的修改也可以調整成單欄或雙欄的橫式版面

我們先將html的基本架構做出來並填入內容
先做出其中一個欄位再複製就好
首先先建立一個wrap區塊、container區塊、item區塊並包含pic圖片區塊(使用假圖)以及txt文字區塊
將圖片跟文字部分都單獨建立區塊後續CSS會比較好操作

接下來開始進入CSS切版調整
一樣是先做出<style>標籤及reset css

接下來要將我們的三個container區塊，放在畫面中間，接著設定container本身的寬度為1200px，並在裡面分成三個item的大小，這邊的寬度大小設為370是因為要加上旁邊的margin各15，剛好會等於每個item 400px的寬度，最後再將三個item排列成橫排，在container中加入display: flex;
