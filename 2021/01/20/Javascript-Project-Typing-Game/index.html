getRandomWord 取得隨機單字函式

這邊在影片課程中是自己設了一個陣列，並透過亂數選擇的方式隨機選取陣列中的單字作為題目，
我在這邊是直接找了一個可以產生隨機文字的API來實作，關鍵字：random word API

這邊亂數的作法跟之前在記帳列表實作中產生亂數產生ID的函式用的方法很像
原理是這樣的：

  1. Math.random()：會隨機產生出0~1之間的小數
  2. Math.floor()：會將小數點捨去並回傳小於等於所給數字的最大整數。
  3. 將亂數產生的小數乘上你需要的倍數，在這裡是陣列中的數量，
  4. 產生的數字來讓我們隨機抓取陣列中的第幾個元素(單字)

之後再將這個單字填入h1中就成為題目了

最後將這整個抓取資料跟填入畫面設為一個函式getRandomWord方便呼叫

Javascript Part 2 - input Event 輸入事件

接著做出計分功能，實際的作法如下：

  1. 先在input綁定事件監聽，這樣輸入時就可以取得輸入的值
  2. 用變數typeWord、randomWord分別放入題目及輸入的值
  3. 利用判斷式 => 若兩者相等則題目刷新且輸入框變回空白
  4. 分數 +1，並將這個值填入畫面中

    Javascript Part 3 - countDown 倒數計時函式

    這部分先加入一個小細節，利用.focus()讓我們進入網頁時直接可以開始遊戲，
    不用手動點擊輸入

再來有了分數同時也要加入時間限制，先將時間利用變數time設為預設值

這邊用到的是setInterval()這個方法來倒計時
這個方法分為兩個參數，第一個參數是要執行的函式，第二個參數則是多久執行一次(毫秒)

所以這邊我們設定time- 1，並將變數的值填入畫面中，並一秒執行一次
所以就會達成畫面顯示每秒減一

在這邊稍微提一下setInterval()可以利用兩種方式來使用，
我上面使用的是將方法封裝到一個函式中，需要的時候呼叫函式來執行。

另外一種則是以下做法：將要執行的函式另外寫，在載入時直接使用方法執行函式。

gameOver 遊戲結束函式

由於setInterval()是會重複執行，所以我們利用判斷式設定一個停止的時機

當倒數到零(time的值等於零)，利用clearInterval()這個方法來停止執行。

結束區塊還包含一個reload按鈕可以重新載入頁面，繼續遊戲。

Javascript Part 4 - Difficulty Part

在難度的部分首先我們要先設定不同難度得分時可以增加的秒數

得分後，利用判斷式來設定：

  1. 若難度為hard => 得分時加兩秒
  2. 若難度為medium => 得分時加三秒
  3. 最後剩下難度為easy => 得分時加五秒

    Difficulty Change Event 難度調整事件

    接著我們要讓重新載入時難度不會變回預設值

我們在調整難度時，將難度的值hard、medium、easy存入localStorage

並利用變數difficulty跟getItem()方法取得存在localStorage的值
並利用判斷式：

  1. 若值不為空，則利用getItem()方法取得存在localStorage的難度值
  2. 否則(值為空)，則值為預設medium
