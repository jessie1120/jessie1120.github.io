HTML

HTML的部分，在選擇電影的地方使用的是select標籤+上option來做出下拉選單的樣式。
同時在每個選項有加入代表票價的值value。

再來在座位選擇的部分，上方用ul跟li的列表作出座位狀態的示意表。
下方則是先放置一個螢幕區塊，再將每一個div區塊做成一個座位，擷取部分程式碼↓

最後是在下方做出顯示目前選擇座位及總計票價的文字敘述，
將之後需要使用javascript進行變更的文字用span標籤包覆並給定id讓後續選取更方便。

CSS

CSS的部分，網頁的整體布局以Flex為主，因為只是需要置中，較為方便。
其中有幾個指令這邊紀錄一下。
appearance:由於不同瀏覽器的支援不同，針對不同瀏覽器的預設樣式修正。
macOS使用的safari需要-webkit-appearance: none;來防止預設的瀏覽器樣式。

針對已選擇座位及未選擇座位，預先設定好CSS讓之後Javascript操作只需要修改className即可。

電影螢幕的部分還有用到一個CSS的3D透視效果perspective，數值越大效果越好。
搭配本身區塊的傾斜transform: rotateX(-45deg);跟陰影box-shadow做出類似電影院螢幕效果。

事件的最後會利用函數將已選擇的數量計算票價更新至下方文字區域。

Javascript Part 2 - Local Storage

這邊主要需要做的功能是，在重整後可以保留之前的選取座位，
將這些內容存在Local Storage本地儲存空間，在需要時可以再拿出來進行運用。
同時也避免關閉分頁後的資料損失。

這邊影片課程給出的功能製作思路↓

  1. copy selected seats into array
    將已選擇座位轉換為陣列array。
  2. map through array
    對陣列中元素進行處理
  3. return a new array indexes
    回傳新陣列中元素的索引值(在陣列中的第幾個位置)

在updateSelectedCount這個function內製作功能

首先利用...運算來將我們選取的位置傳入[]成為陣列array，
接著利用map()這個method處理陣列中的元素，
返回我們選取的位置在陣列中的索引值(在陣列中哪個位置，從零開始，利用indexOf()方法)。

接下來將我們剛剛取得的索引值存入localStorage，
這邊第一個參數作用是給定名稱，稱為keyName(鍵名) => 下面這個例子叫做selectedSeats
第二個參數作用是給定值，稱為keyValue(鍵值)也就是這個keyName對應的值。
但keyName和keyValue都必須是字串型別，所以用JSON.stringify()將陣列轉成字串型別。

setMovieData() 更新電影資料函式

再來我們除了選擇座位需要存入localStorage，選擇哪一部電影(票價)也需要存入。

這樣一樣利用keyName+keyValue來存入localStorage，但由於這次的keyValue本身即為字串，
所以不用再使用JSON.stringify()進行轉換。

接著在電影選擇事件中加入此函式，這邊的selectedIndex就是取得select標籤的索引值(第幾項)
同時e.target.selectedIndex = 選擇哪一部電影，e.target.value = 該部電影票價。
這兩個東西在呼叫此函式的時候，會存入localStorage。

Javascript Part 3 - Populate UI With Saved Data

populateUI() 回填函式

這個函式呼叫的位置會放置在宣告變數之後，讓它可以第一個被載入，將存放的資料回填。
這個函式的主要功能是將剛剛存入localStorage的座位資料，重新放回顯示區域(畫面上)，流程如下↓

  1. 首先我們將剛剛存入localStorage的selectedSeats從字串重新轉回array
    這邊用到的是JSON.parse()這個方法=>將剛剛setItem存入localStorage的東西取出來。

  2. 開始第一次判斷，條件為selectedSeats不為空值，且至少須有一個在localStorage中。

  3. 第二次判斷：若通過條件，利用forEach()開始檢查selectedSeats的索引值是否存在陣列中，
    這邊的條件 (> -1) 是因為indexOf()這個方法確認陣列內沒有這個元素時會回傳索引值= -1。

  4. 通過第二次判斷之後，將class屬性+上selected

接下來做一樣的事，將電影選擇的資料取出，不為空值的話，放回movieSelect

最後在整個js的最下方呼叫一次updateSelectedCount() 函式，
讓我們重整時不用經過事件觸發，可以直接更新下方文字敘述。
