專案實作功能說明

這次實作的專案主要功能是倒數計時到隔年的新年(一月一號零點)為止

實際需要實做的功能：

  • 取得現在時間時間並計算到達目標的時間
  • 將倒計時結果呈現在畫面上
  • 重整頁面時，加入載入圖片並隱藏倒計時

Javascript Part 1 - showCountdown 倒計時函式

首先我們透過變數及Date()方法來取得目前年份currentYear及目標時間newYearTime

接著利用showCountdown 倒計時函式來計算目前時間與目標時間的差距並新增到畫面上

  1. 先利用Date()方法取得目前時間
  2. 計算目前時間與目標時間的差距並填入變數countdownTime中(這邊取得的數字單位是毫秒)
  3. 利用Math.floor()方法來取整數：將小數點後無條件捨去
    舉個例子來說明這邊的計算：其他依序增加須除的數及取餘數的值
    seconds的值 = 毫秒/1000，Math.floor()取整數後，利用%取餘數，得到小於60的值 = 秒
  4. 接著將我們取得的值填入畫面(DOM)中
  5. 呼叫函式顯示在畫面上

    addStandardTime 格式化時間函式

    這邊用一個小小的判斷式來處理時間的格式，當值小於 10 時，在數字前補個零。

    Refresh countdown time

    再來我們需要這個函式每秒更新一次，利用setInterval()方法每秒執行一次。 並且在標題內加入倒數年份，讓我們可以不用每年更新一次，變成動態的更新目前目標時間。
